1. ENUM

ENUM is a data type in MySQL that lets you define a list of allowed string values for a column.

Example:

CREATE TABLE members (
    member_id INT AUTO_INCREMENT PRIMARY KEY,
    membership_type ENUM('Basic', 'Premium', 'VIP') NOT NULL
);


Here, membership_type can only be 'Basic', 'Premium', or 'VIP'.

If you try to insert something else, MySQL will throw an error.
